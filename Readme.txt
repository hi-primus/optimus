Optimus is the missing framework to profile, clean, process and do ML in a distributed fashion using Apache Spark (PySpark).

- Requirements:

* Apache Spark>=2.3.0
* Python>=3.6

- Installation:

In your terminal just type:

$ pip install optimuspyspark
